token API¶

API Contents¶

GET /api/token¶

Monitor 서비스 API 리소스 접근을 위한 토큰 가져오기

요청 예:

GET /api/token HTTP/1.1
Host: example.com
Authorization: Basic dGVzdHVzZXI6dGVzdHBhc3N3ZA==

응답 예:

HTTP/1.1 200 OK
Content-Type: application/json

{
    "duration": 600,
    "token": "eyJhbGciOiJIUzI1NiIsImV4cCI6MTM5MzgxNDMxOSwiaWF0IjoxMzkzODEzNzE5fQ.eyJ1c2VybmFtZSI6IkdNMTMwMjQwNyJ9.c9tfGYZOqNn83zbRqlXCqP2IH9FLFeA29EPnVktpCp0"
}
Request Headers:
 
  • Authorization -- Basic "[ixmonitor계정:ixmonitor비번] base64 encoding 스트링"
Response Headers:
 
  • Content-Type -- json만을 지원
Status Codes:
  • 200 OK -- no error
  • 401 Unauthorized -- 잘못된 토큰 or 시간 오류
  • 403 Forbidden -- 계정 정보 오류
